# Projet_js (site vitrine)

Vite est utilisé ici pour :
-créer un serveur de développement rapide avec rechargement automatique  
-simplifier l’organisation du projet en modules ES  
-générer un build optimisé pour la production  

# Objectif du projet

-Le site vitrine est là pour un objectif de présenter l’univers, les mécaniques et l’ambiance de l’Idle Game “INTERN GRIND”.

-Il permet d’expliquer au joueur le concept du jeu, ses objectifs, son style humoristique, ainsi que les différentes ressources à accumuler et les niveaux à débloquer.

-Le site sert de porte d’entrée vers le jeu : il donne envie de jouer, contextualise l’histoire et fournit une présentation claire et accessible. Il fonctionne comme un support visuel et informatif destiné aux visiteurs, aux testeurs et aux encadrants du projet.


# Prérequis :

Avant de commencer, installez :

- **Node.js** (version 18 ou plus)
- **npm** (inclus avec Node.js)

Aucune installation supplémentaire n’est nécessaire :  

toutes les dépendances du projet, dont **Vite**, 
seront automatiquement installées via : npm install

et après (exécution): npm run dev;
